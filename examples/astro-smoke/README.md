# Sygnal Astro Smoke Test

Minimal Astro app to verify the `sygnal/astro` integration.

## Run

```bash
cd /Users/troy/Documents/Displera/sygnal/examples/astro-smoke
npm install
npm run dev
```

Open the dev server URL and click the **Increment** button.  
If the count updates, client hydration through the Sygnal Astro plugin is working.
